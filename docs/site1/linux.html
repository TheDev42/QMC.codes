Getting Started with Linux

This guide covers essential Linux commands for managing your self-hosted server. These commands work on most Debian-based distributions (Ubuntu, Debian) and many other Linux distros.

Tip: Practice these commands in a safe environment before using them on a production server.

Package Management

Update Package List

Refresh the list of available packages from repositories

sudo apt update

Upgrade All Packages

Install latest versions of all installed packages

sudo apt upgrade -y

Install Package

Install a new package

sudo apt install nginx

Remove Package

Remove an installed package

sudo apt remove nginx

Full System Upgrade

Complete system upgrade including kernel

sudo apt dist-upgrade -y

File and Directory Operations

List Files

List all files in current directory

ls -lah

Change Directory

Navigate to a directory

cd /var/www/html

Create Directory

Create a new directory

mkdir myproject

Copy File

Copy a file

cp source.txt destination.txt

Move/Rename File

Move or rename a file

mv oldname.txt newname.txt

Remove File

Delete a file

rm file.txt

Remove Directory

Force remove a directory and contents

rm -rf myfolder

Viewing and Editing Files

View File Content

Display contents of a file

cat file.txt

View File Page by Page

View file with pagination

less file.txt

Edit with Nano

Edit file using nano (beginner-friendly)

sudo nano /etc/nginx/nginx.conf

Edit with Vim

Edit file using vim

sudo vim /etc/nginx/nginx.conf

Find Files

Search for files by name

find /home -name "*.txt"

Grep Search

Search for text in files

grep "search term" file.txt

System Information

System Information

Display system information

uname -a

Disk Usage

Show disk space usage

df -h

Memory Usage

Display memory usage

free -h

CPU Usage

View real-time CPU and process info

top

System Uptime

Show how long system has been running

uptime

Process Management

List Processes

List all running processes

ps aux

Kill Process

Terminate a process by PID

kill -9 [PID]

Kill Process by Name

Kill process by name

pkill nginx

Service Management

Check Service Status

Check if a service is running

sudo systemctl status nginx

Start Service

Start a service

sudo systemctl start nginx

Stop Service

Stop a service

sudo systemctl stop nginx

Restart Service

Restart a service

sudo systemctl restart nginx

Enable Service on Boot

Start service automatically on boot

sudo systemctl enable nginx

Networking Commands

Check IP Address

Display network interface IP addresses

ip addr

Check Open Ports

List all listening ports

sudo netstat -tulpn

Ping Test

Test network connectivity

ping -c 4 google.com

Trace Route

Show route to destination

traceroute google.com

Test Port

Test if port is open

nc -zv example.com 80

File Permissions

Change Permissions

Set file permissions (rwxr-xr-x)

chmod 755 script.sh

Change Owner

Change file/directory ownership

sudo chown www-data:www-data /var/www

User Management

Create User

Create a new user

sudo adduser username

Add User to Sudo

Give user sudo privileges

sudo usermod -aG sudo username

Logs and Monitoring

View System Logs

Follow system log in real-time

sudo tail -f /var/log/syslog

View Nginx Logs

View Nginx access logs

sudo tail -f /var/log/nginx/access.log

Archives and Compression

Create Archive

Create compressed tar archive

tar -czvf archive.tar.gz /folder

Extract Archive

Extract compressed archive

tar -xzvf archive.tar.gz

Download Files

Download with Wget

Download file from URL

wget https://example.com/file.zip

Download with Curl

Download file using curl

curl -O https://example.com/file.zip
