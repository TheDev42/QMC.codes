What is Proxmox?

Proxmox VE (Virtual Environment) is an open-source server virtualization platform that combines KVM hypervisor and LXC containers in a single, easy-to-use solution.

Key Concept: Proxmox uses a web-based interface for managing virtual machines and containers, making it accessible even for beginners.

Installing Proxmox VE

System Requirements

  • CPU: 64-bit (Intel VT-x/AMD-V)
  • RAM: Minimum 2GB (recommended 4GB+)
  • Disk: Minimum 32GB
  • Network: Ethernet adapter

Download and Install

  1. Download ISO from proxmox.com
  2. Burn ISO to USB or DVD
  3. Boot from media and follow installer wizard
  4. Configure network settings during installation
  5. Set root password and email for notifications
  6. Access web interface at https://your-ip-address:8006

Basic Configuration

Update System

Update Proxmox packages

apt update && apt upgrade -y

Remove Enterprise Repository

Remove trial subscription message

rm /etc/apt/sources.list.d/pve-enterprise.list

Add No-Subscription Repository

Add useful testing repository

echo "deb http://download.proxmox.com/debian/pve bookworm pve-no-subscription" | tee /etc/apt/sources.list.d/pve-no-subscription.list

Creating Virtual Machines

Create VM via Command Line

Create QEMU/KVM virtual machine

qm create 100 --name "my-vm" --memory 2048 --cores 2 --net0 virtio,bridge=vmbr0 --ostype l26 --ide0 local-lvm:vm-100-disk-0,size=20G

List All VMs

Show all virtual machines

qm list

Start VM

Boot up virtual machine

qm start 100

Stop VM

Shut down virtual machine

qm stop 100

Remove VM

Delete virtual machine

qm destroy 100

Show VM Config

View VM configuration

qm config 100

Clone VM

Clone existing VM

qm clone 100 101 --name new-vm

Creating Containers (LXC)

Lightweight Linux containers use less resources than full VMs

Create Container

Create LXC container

pct create 200 local:vztmpl/debian-12-standard_amd64.tar.zst --rootfs local-lvm:10G --hostname my-container --password secret --net0 name=eth0,bridge=vmbr0

List Containers

Show all LXC containers

pct list

Start Container

Launch container

pct start 200

Stop Container

Shutdown container

pct stop 200

Enter Container

Access container shell

pct enter 200

Storage Management

List Storage

Show available storage

pvesm status

Add NFS Storage

Mount network storage

pvesm add nfs shared-nfs --server 192.168.1.100 --export-path /mnt/share

Cluster Management

Note: Proxmox supports clustering multiple nodes together for high availability.

Cluster Status

View cluster health

pvecm status

List Nodes

Show all nodes in cluster

pvesh get /nodes

Backups and Disaster Recovery

Backup VM

Create backup of VM

vzdump 100 --storage local --mode snapshot

Restore VM

Restore from backup

qmrestore vzdump-qemu-100.vma 100 --storage local
